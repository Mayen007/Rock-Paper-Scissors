# 🎮 Rock Paper Scissors Game

## 🎮 Overview

A **premium, feature-rich** version of the classic **Rock, Paper, Scissors** game built with **HTML**, **CSS**, and **JavaScript**. Experience stunning visual effects, comprehensive statistics tracking, and smooth animations while playing against an intelligent computer opponent. Features both casual play and auto-play modes with complete data persistence.

---

## ✨ Features

### 🎮 **Core Gameplay**

- **Interactive Controls**: Play by clicking buttons or using keyboard shortcuts
- **Smart Computer AI**: Randomized computer moves for fair gameplay
- **Real-Time Feedback**: Instant results with visual and statistical updates

### 🎨 **Visual Effects** _(NEW!)_

- **Particle Systems**: Explosive particle effects for wins and losses
- **Confetti Celebrations**: Spectacular confetti rain for big wins (5+ streaks)
- **Enhanced Animations**: Glowing effects, smooth transitions, and move-specific animations
- **Dynamic Button Effects**: Radial glow effects and hover animations
- **Winner/Loser Indicators**: Visual feedback showing winning and losing moves

### ⌨️ **Keyboard Shortcuts**

- `R`: Select **Rock**
- `P`: Select **Paper**
- `S`: Select **Scissors**
- `A`: Toggle **Autoplay mode**
- `Backspace`: **Reset** the score (with confirmation)

### 🔁 **Autoplay Mode**

Watch the computer play automatically — perfect for testing game dynamics and enjoying the visual effects.

### 📊 **Advanced Statistics**

Track your performance with:

- **Win Rate Percentage**: Real-time calculation of your success rate
- **Current Streak**: Track your ongoing win/loss/tie streaks
- **Best Streak**: Remember your longest winning streak
- **Total Games**: Complete game counter with persistent storage

### 📋 **Game History**

- **Last 10 Games**: Detailed history with moves, results, and timestamps
- **Visual Indicators**: Color-coded results with emoji feedback
- **Persistent Storage**: All history saved across browser sessions
- **Real-Time Updates**: Automatic history updates after each game

### 🌓 **Theme System**

- **Dark/Light Modes**: Switch themes for comfortable gameplay
- **Consistent Styling**: Seamless theme integration across all components
- **User Preference Memory**: Theme choice saved automatically

### 📱 **Responsive Design**

- **Mobile Optimized**: Perfect gameplay on all screen sizes
- **Adaptive Layouts**: Smart component repositioning for different devices
- **Touch-Friendly**: Large, accessible buttons for mobile users
- **Smart Shortcuts Panel**: Visible on desktop, hidden on mobile for clean UI

### 🔒 **User Experience**

- **Confirmation Modals**: Prevent accidental data loss
- **Smooth Animations**: Professional-grade transitions and effects
- **Accessible Design**: High contrast, readable fonts, and clear navigation
- **Performance Optimized**: Efficient particle systems and smooth 60fps animations

### 💾 **Data Persistence**

All game data automatically saved:

- Game statistics and scores
- Complete game history
- User preferences and themes
- Streak records and achievements

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Mayen007/rock-paper-scissors.git
cd rock-paper-scissors
```

### 2. Open the Game

Simply open the `index.html` file in your preferred browser:

- **Windows:**

  ```bash
  start index.html
  ```

- **macOS:**

  ```bash
  open index.html
  ```

- Or manually double-click `index.html` from your file explorer.

### 3. Start Playing!

- Click the move buttons or use keyboard shortcuts
- Watch the stunning visual effects
- Track your progress in the statistics section
- Challenge yourself to beat your best streak!

## 🎯 Game Rules

- **Rock** beats **Scissors**
- **Paper** beats **Rock**
- **Scissors** beats **Paper**
- Same moves result in a **Tie**
